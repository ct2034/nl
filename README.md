# RasPiO Night Light

This Github repo holds the example sketch for the RasPiO Night Light.

https://rasp.io/nightlight

## Arduino IDE settings
* Board: Arduino Nano
* Processor: ATmega328P (old Bootloader) 

## Used PINs
* A7 (in) - LDR
* D2 (out) - Buzzer
* D3 (in) - Switch (NEW)
* D5 (out) - clock FastLED
* D7 (out) - data FastLED
* D12 (in) - PIR
* D13 (out) - LED
